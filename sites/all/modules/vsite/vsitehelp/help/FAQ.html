Q: What the is Scholars' Web Sites Project?
A: The Scholars' Web Sites Project is a highly customizable web site building and content management tool intended for academic scholars, faculty and graduate students.

Back to top

Q: Who is eligible to get a site?
A: The Scholars' Web Sites Project is intended for Harvard affiliated scholars, faculty, and graduate students. You must have a valid Harvard ID in order to get a site.

Back to top

Q: How do I create a web site?
A: You can get your web site by going to the Scholars' Web Sites Project front page and clicking the 'Get Your Web Site!' button. Submit your information in the form that appears and your should see a confirmation that your web site has been created.

Back to top

Q: Do I need any special programs?
A: No, you can everything related to your site using your web browser.

Back to top

Q: Do I need to know HTML or other web technologies?
A: While it might be helpful to know HTML or other web technologies, it is not required. A basic understanding of the web is all that is needed. The software will do the rest.

Back to top

Q: Can I create a site for my department/center/project?
A: Yes, there is a special "Project" installation of Scholars' Web Sites Project which is designed for this purpose.

Back to top

Q: Where are the web sites hosted?
A: The Scholars' Web Sites Project web sites are hosted by Harvard MIT Data Center at Harvard University.

Back to top

Q: Can I design the look and layout of my site?
A: Yes, the Scholars' Web Sites Project comes with easy to use theming and layout tools which allow you design a unique and elegant web site.

Back to top

Q: Can I upload files to my site?
A: Yes, you can upload files (documents, images, etc.), to your site directly from your web browser.

Back to top

Q: What happens to my site if I leave Harvard?
A: Your site will still remain active after your leave Harvard unless you request to have it removed.
