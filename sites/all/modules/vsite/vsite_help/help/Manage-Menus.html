This section shows you what menu entries are avalible for your site and allows you to select where they will show.

The Vertical list along the left side shows you the avalible menu "links", if you create custom menu links (see Creating custom menu links) they will also show here.  Some columns may be greyed out, this means that the feature that goes with this menu "link" is not enabled.  Features can be enabled under the "Features" tab in settings.

The horizontal list across the top lists the avalible menus as well as "none" which will hide that link.  Additionally there is a column called "delete", this allows you to remove custom menu entries from the list.  If an entire column for a menu is greyed out then the menu has not been put into the site layout.  You can drag the menu into the site layout on the "Layout" tab under "Settings".
